* Connecting to the ASMI

** Connect using DHCP

** Connect using Static IP

* Enabling IPMI

* Powering on your server with IPMI

** Connecting from Linux notebook

** Connecting from Windows notebook

* Configuring Petitboot and installing {rel8}
